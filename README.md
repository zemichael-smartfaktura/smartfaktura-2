# SmartFaktura 2.0
Goal:
Build a modern, multi-tenant invoicing SaaS.

Design target:
EU-level quality and compliance.

Initial market:
Africa.

Long-term expansion:
EU.

Principles:
- One senior developer
- MVP first
- Clean architecture
- No legacy code
- Full ownership
- ## Scope MVP
- Multi-tenant (fra dag 1)
- Faktura
- Kunder
- Produkter/tjenester
- PDF eksport

## Out of scope (for MVP)
- Stripe
- Skatt/regnskap per land
- Avansert rapportering
