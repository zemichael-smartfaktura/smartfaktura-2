# SmartFaktura 2.0

Goal:
Build a modern, multi-tenant invoicing SaaS.
Initial market: EU.
Expansion: Africa.
Principles:
- One senior developer
- MVP first
- Clean architecture
- No legacy code
- Full ownership
- ## Scope MVP
- Multi-tenant (fra dag 1)
- Faktura
- Kunder
- Produkter/tjenester
- PDF eksport

## Out of scope (for MVP)
- Stripe
- Skatt/regnskap per land
- Avansert rapportering
