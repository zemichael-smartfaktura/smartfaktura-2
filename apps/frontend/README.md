# Frontend

React + Vite app. Run from repo root or from this directory.

**From root:** `bun run dev:app`

**From here:**

```bash
cd apps/frontend
bun install
bun run dev
```

| Command           | Description   |
|-------------------|---------------|
| `bun run dev`     | Dev server    |
| `bun run build`   | Production    |
| `bun run preview` | Preview build |
