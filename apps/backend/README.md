# Backend

Express API. Run from repo root or from this directory.

**From root:** `bun run dev:api`

**From here:**

```bash
cd apps/backend
bun install
bun run dev
```

| Command         | Description      |
|-----------------|------------------|
| `bun run dev`   | Dev (watch)      |
| `bun run build` | Build to `dist/` |
| `bun run start` | Run built output |
